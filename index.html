Why Backit?

Plug-and-Play

Install LSPs with a single command. No configs. No hassle.

Language Agnostic

Supports LSPs for JavaScript, Python, Rust, Go, and more.

Fast & Minimal

Built to be fast, simple, and extendable. Nothing more.

Supported LSPs

💡 Pyright
⚙️ rust-analyzer
🌐 tsserver
🚀 gopls
📦 bash-language-server
🔧 lua-language-server

CLI Preview

$ backit init
$ backit install pyright
$ backit install rust-analyzer
$ backit list
$ backit switch pyright@1.1.300
$ backit update
